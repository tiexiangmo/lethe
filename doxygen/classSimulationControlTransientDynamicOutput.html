double SimulationControlTransientDynamicOutput::calculate_time_step ( )
overrideprotectedvirtual

Calculates the next value of the time step. The time step is calculated in order to ensure that the CFL condition is bound by the maximal CFL value. The new time step is equal to adaptative_time_step_scaling * the previous time step. If this surpasses the simulation time, the output_time or if it surpasses the maximal CFL value, the time step is scaled down to ensure that these elements are respected.
