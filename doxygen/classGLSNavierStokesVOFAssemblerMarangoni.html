Detailed Description

template<int dim>
class GLSNavierStokesVOFAssemblerMarangoni< dim >

Class that assembles the marangoni effect for the Navier-Stokes equations according to the following equation:

\(\mathbf{F_{Ma}}= \frac{\partial \sigma}{\partial T} \left[ \nabla T - \frac{\nabla \phi}{| \nabla \phi |} \left( \frac{ \nabla \phi } {| \nabla \phi |} \cdot \nabla T \right) \right] | \nabla \phi | \frac{2 \rho}{\rho_0 + \rho_1} \)

Template Parameters
dimAn integer that denotes the number of spatial dimensions
