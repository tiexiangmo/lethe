Detailed Description

template<int dim, int spacedim = dim>
class GridMotion< dim, spacedim >

Carries out motion of the triangulation, including rotational, translational and rotational-translational motions.

Note
Grid motion with a ParsedFunction should be added to the code as a future improvement
