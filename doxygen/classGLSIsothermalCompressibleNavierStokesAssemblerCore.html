Detailed Description

template<int dim>
class GLSIsothermalCompressibleNavierStokesAssemblerCore< dim >

Class that assembles the core of the isothermal compressible Navier-Stokes equations. According to the following weak form: \(\nabla \cdot (\rho \mathbf{u}) + \rho \mathbf{u} \cdot \nabla \mathbf{u} - \nabla p - \mu \nabla^2 \mathbf{u} = 0\) with a full GLS stabilization including the laplacian of the test function.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
