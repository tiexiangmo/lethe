Detailed Description

template<int dim>
class GLSVansAssemblerVorticalTorque< dim >

Class that assembles the Viscous dissipative torque due to fluid vorticity as defined by Derksen (2004). M_viscous_vorticity = pi * pow(dp, 3) * mu * (0.5 * vorticity)

The complete model described by Derksen is composed of GLSVansAssemblerViscousTorque + GLSVansAssemblerVorticalTorque
