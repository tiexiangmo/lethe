vector_value. This vector_value function is used to construct the geometry manifold. It changes the new point (np) of the hill grid to the original point (op) of the transformed hyper_rectangle grid. This function is mandatory to use FunctionManifold. First, it finds the minimum value of y depending the x position and then calculates the op with the inverse of the transformation done by hill_geometry function.

Parameters
np.A point in space.
values.The vector of values which will be calculated at the position p.
