Detailed Description

template<int dim>
class CahnHilliardAssemblerCore< dim >

Class that assembles the core of the Cahn-Hilliard equation with the following weak form: dPhi/dt + u * gradPhi = div(M(Phi)*grad eta) eta - f(Phi) + epsilon^2 * div(grad Phi) = 0.
